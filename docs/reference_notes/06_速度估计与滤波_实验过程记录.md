# 速度估计与滤波 实验过程记录

## 1. 实验目的

本次实验主要验证电机速度估计结果经过不同滤波方法处理后，对启动、加速、减速和停转过程的影响，并通过串口导出的 CSV 数据做对比分析。

## 2. 实验平台

- 控制板：STM32G4 + MCSDK 工程
- 速度估计基线：`STO + PLL`
- 数据采集方式：串口 CSV
- 可视化方式：本地 HTML 页面

## 3. 实验方案

本课设后续建议保留两类实验：

- 主实验：`0 -> 500 rpm`，用于看阶跃响应
- 调速实验：`0 -> 500 rpm -> 1000 rpm -> 500 rpm -> stop`，用于看动态调速和滤波差异

本轮采用的调速实验节拍是：

- `500 rpm` 维持 `6 s`
- 提速到 `1000 rpm` 维持 `6 s`
- 再降回 `500 rpm` 维持 `6 s`
- 最后停转

滤波方法共 7 组：

1. `NONE`
2. `LPF1 shift2`
3. `LPF1 shift3`
4. `LPF1 shift4`
5. `MOVAVG8`
6. `WMA4`
7. `ADALPF`

## 5. 实验现象

1. 启动阶段波动最大，主要原因是电机从静止加速到目标转速，曲线本身处于爬升过程。
2. `6-12s` 和 `12-18s` 两段的均值基本接近目标值，说明各组都能正常跟踪转速变化。
3. `MOVAVG8` 和 `WMA4` 在停转尾段更短，`<=30 rpm` 的时间约 `420 ms`。
4. `LPF1 shift3` 和 `LPF1 shift4` 的综合表现比较平衡，既能平滑速度，又不会明显拖慢响应。
5. `ADALPF` 这版没有明显优于固定参数低通，说明自适应阈值还可以继续优化。

## 6. 初步分析

本轮数据说明：

- 速度滤波确实会影响控制效果，因为滤波后的速度值会重新送回速度环。
- 滤波越强，稳态通常越平滑，但动态响应会更慢。
- 这轮实验里，各方法在中段平均速度差别不大，主要差异体现在曲线平滑程度和停转尾段。

从当前结果看，`LPF1 shift3` 更适合作为基础方案；`MOVAVG8` 和 `WMA4` 适合做对比展示；`ADALPF` 还有继续优化空间。

## 7. 实验结论

本次实验完成了 7 组滤波对比，数据采集和可视化流程正常，新的 `500 -> 1000 -> 500 -> stop` 节拍也运行稳定。当前工程已经具备后续继续做滤波优化和结果展示的基础。
