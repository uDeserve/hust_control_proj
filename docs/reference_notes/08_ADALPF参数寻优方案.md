# ADALPF 参数寻优方案

## 目标

先把自适应滤波调到一个“答辩时讲得通、曲线也看得过去”的版本，再做重复性实验。

## 当前基线

- 快档：`shift=2`
- 慢档：`shift=4`
- 进入强滤波门限：`18 rpm`
- 退出强滤波门限：`40 rpm`
- 稳态确认次数：`64`

## 推荐调参顺序

### 第一轮

先只改门限，不动 `shift`：

- `enter_rpm = 12 / 18 / 24`
- `exit_rpm = 32 / 40 / 48`

目的：

- 让启动阶段尽量快
- 让稳态阶段尽量稳
- 看切换是否过于频繁

### 第二轮

再改确认次数：

- `confirm_n = 32 / 64 / 96`

目的：

- 次数太小，容易来回切
- 次数太大，稳态切换太慢

### 第三轮

最后再看快慢档位：

- `fast_shift = 1 / 2`
- `slow_shift = 3 / 4`

目的：

- 看启动响应和稳态平滑能不能两头兼顾

## 评价标准

每轮参数都看下面四点：

- 启动是否明显拖慢
- 1000 rpm 段是否更稳
- 500 rpm 回切时是否更平滑
- 停转尾段是否更短

## 建议最终选择

优先选：

- 启动不肉
- 稳态不抖
- 参数解释简单

也就是尽量选一个能在答辩时一句话讲明白的方案。
