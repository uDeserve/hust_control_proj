# 速度滤波 MATLAB 仿真说明

## 1. 目的

这个目录用于补课设中的“先仿真，再实操，再复盘，再调优”流程。

当前仿真不试图完整复现 MCSDK 的 FOC 和 `STO + PLL` 底层估计器，而是针对本课设代码实际改动的部分：速度估计结果进入速度环之前的滤波处理。

也就是说，仿真关注的问题是：

- 如果原始估计速度带有噪声和纹波，不同滤波方法的输出有什么差别。
- 强滤波为什么会带来滞后。
- 弱滤波为什么稳态更抖。
- 自适应低通为什么有机会兼顾动态响应和稳态平滑。

## 2. 文件

- `speed_filter_sim_main.m`：主仿真脚本，直接运行。
- `sim_outputs/`：脚本运行后自动生成，保存图片和指标 CSV。

## 3. 运行方式

在 MATLAB 里打开本目录，然后运行：

```matlab
speed_filter_sim_main
```

脚本会自动生成两类场景：

- `step_0_500`：主实验，对应 `0 -> 500 rpm` 阶跃响应。
- `profile_0_500_1000_500`：调速实验，对应 `0 -> 500 -> 1000 -> 500 -> stop`。

每个场景会输出：

- 对比曲线 PNG
- 指标 CSV

## 4. 和板上实验的对应关系

仿真中的方法名称和固件/CSV 保持一致：

- `NONE`
- `LPF1_shift2`
- `LPF1_shift3`
- `LPF1_shift4`
- `MOVAVG8`
- `WMA4`
- `ADALPF_2to4_h18_40_n32`

其中 `ADALPF_2to4_h18_40_n32` 对应目前 ADALPF 参数寻优后较好的候选。

## 5. 汇报时怎么讲

建议在 PPT 里这样使用：

1. 先展示仿真曲线：说明不同滤波器的理论趋势。
2. 再展示实测曲线：说明真实电机上是否符合仿真趋势。
3. 最后讨论差异：真实系统会受到启动过程、负载、采样周期、估计器误差等影响。

这样比直接展示实测曲线更完整，也更符合老师建议的工程流程。
