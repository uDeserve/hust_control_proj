# 电机速度 CSV 可视化

这个工具是一个离线本地页面，用来对比不同滤波方法、不同参数下的实验曲线，重点适合你现在这种课程设计实验。

## 现在已经支持

- 一次导入多个 CSV 文件
- 拖拽 CSV 到页面导入
- 同一个 CSV 内多列直接叠加对比
- `主 Y 轴列 + 额外叠加列 + 第二 Y 轴列` 三路组合
- 快速预设：
  - `原始 vs 滤波`
  - `滤波 vs 目标`
  - `启动放大`
  - `稳态波动`
- 每个数据集单独开关显示
- 每个数据集可填写备注
- 图例自动翻译成“方法名 + 中文解释”
- 支持 `单图堆叠 / 多图并列` 两种展示模式
- 单图堆叠模式下，目标速度只保留一条统一虚线
- 多图并列模式下，每张图都是“该方法实际速度 + 目标速度”同图显示
- 统计表自动输出：
  - 点数
  - 均值
  - 最小值
  - 最大值
  - 峰峰值
  - 标准差
- 实验指标表自动输出：
  - 上升时间
  - 稳态建立时间
  - 稳态均值
  - 稳态误差
  - 稳态标准差
  - 稳态峰峰值
  - 最大超调
  - 总时长
- 导出 PNG 图片

## 用法

1. 打开 [index.html](/mnt/e/hermes/mc_speed_viewer/index.html)
2. 点击 `导入 CSV`，或者直接把 CSV 拖到页面左侧导入框
3. 先选列：
   - `X 轴列`：一般选 `time_ms`
   - `主 Y 轴列`：一般选 `raw_speed_rpm` 或 `filtered_speed_rpm`
   - `额外叠加列`：一般选另一个速度列
   - `第二 Y 轴列`：一般选 `target_speed_rpm`
4. 如果是常规速度实验，直接点：
   - `原始 vs 滤波`
   - `滤波 vs 目标`
5. 如果要看动态细节，点：
   - `启动放大`
   - `稳态波动`
6. 如果想看整体对比：
   - `图表模式` 选 `单图堆叠`
7. 如果想让每种方法各占一张小图：
   - `图表模式` 选 `多图并列`
8. 看主图/小图和下面统计表
9. 如果是整次实验对比，继续看最下面的 `实验指标对比` 表
10. 满意后点 `导出 PNG`

## 推荐 CSV 列

建议你后续实验尽量统一成这些列名，后面比较省事：

- `time_ms`
- `target_speed_rpm`
- `raw_speed_rpm`
- `filtered_speed_rpm`
- `control_output`
- `method_name`
- `param_tag`

## 你现在最适合的对比方式

- 同一滤波方法，不同参数：
  - 多个 CSV 一起导入
  - `主 Y 轴列` 选 `filtered_speed_rpm`
  - `第二 Y 轴列` 选 `target_speed_rpm`
- 不同滤波方法，同一工况：
  - 多个 CSV 一起导入
  - 用 `启动放大` 看动态响应
  - 用 `稳态波动` 看抖动大小
  - 再看 `实验指标对比` 表里的上升时间、稳态误差和稳态标准差
- 同一个实验里直接比较原始速度和滤波后速度：
  - 点 `原始 vs 滤波`
